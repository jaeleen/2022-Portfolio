Project: Netflix Clone (HTML & CSS)

I developed a fully responsive clone of the Netflix landing page using only HTML and CSS. This project focused on replicating the core visual and structural elements of the Netflix interface, including the layout, typography, and media display features. The goal was to enhance my front-end development skills by creating a pixel-perfect clone that closely resembled the real Netflix homepage.

Key Features

Responsive Design: Used CSS Flexbox and media queries to ensure the site adapts seamlessly across devices (mobile, tablet, desktop). Hero Section: Created a dynamic hero section with a background image and overlay text, closely mimicking Netflix’s design. Content Layout: Built horizontal, scrollable content carousels using Flexbox, displaying movie/show thumbnails similar to Netflix. Navigation Bar: Designed a responsive navbar with logo placement, navigation links, and a sign-in button, maintaining consistency across all screen sizes. Technologies Used HTML5: Structured the layout and content. CSS3: Styled the project, focusing on layout and responsiveness. Google Fonts: Used for matching Netflix’s typography. Results The Netflix clone showcases my ability to create responsive and visually accurate designs using only HTML and CSS. It highlights my attention to detail and proficiency in front-end development.
